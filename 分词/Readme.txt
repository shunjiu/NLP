1.数据存储：
1）为方便查找，使用了二维字典
2）二维字典仅保存出现前后关系的两个词
3）标点后面的词都按开头算,二维字典有个'start'的key,记录开头的词表
4）每个字典里面有个total项，值为表格每一行所有数值的和，方便计算

2.数据切分：
1)使用递归切分，切分时保留当前切分结果以及父节点
2)每一句用$表示结束，当结束时从结束位置向前递归寻找父节点，最终得到所有切分结果
3)当只剩一个字且不在字典中（未登录词）直接把单个字加入切分结果

3.极大似然估计
1）使用laplace平滑
2）处理所有切分可能时，若字典中没有当前组合，直接1/total

4.测试集处理
1）以标点断句
2）句子长度大于25时切成两半，之前一句话40多字，跑了两个小时没有结果，现在不到一分钟
